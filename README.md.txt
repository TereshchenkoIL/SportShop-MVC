# SportShop-MVC

### Description

ASP NET MVC Core application for simple sneakers store which implies CRUD operations.

![Alt text](/MySportShop/wwwroot/images/home.png "Home")

### Functionality
* User
    * Register
    * Login
    * Logout
    * Order Sneakers
    * View product details
    * Add product to cart
    * Cancel Order
    * Edit Order
    * Change Password
    * Filter sneakers
